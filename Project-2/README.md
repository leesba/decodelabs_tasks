## Key Concepts Covered

- **IPO Model** — Input → Process → Output
- **HTTP Methods** — GET (Safe, Idempotent) · POST (Creation) · PUT (Replacement) · DELETE (Removal)
- **RESTful Conventions** — Resources are Nouns, Methods are Verbs
- **JSON** — Keys as strings, no trailing commas, nested objects & arrays
- **Status Code Families** — 2xx Success · 4xx Client Error · 5xx Server Error
- **Validation** — Syntactic (format) + Semantic (logic) + AuthN/AuthZ
- **Live Fetch Demo** — GET & POST requests with real API responses
