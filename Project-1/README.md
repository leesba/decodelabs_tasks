## Key Concepts Covered

- **Semantic HTML5** — Proper landmarks (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- **CSS Grid & Flexbox** — Grid for macro layout, Flexbox for micro alignment
- **Mobile-First Design** — Responsive breakpoints at 768px and 1024px
- **CSS Custom Properties** — Design tokens for colors, spacing, typography
- **Fluid Typography** — `clamp()` for scalable headings
- **Vanilla JavaScript** — Hamburger nav, dynamic year, form validation
